# HuntCozy

HuntCozy is an Android app that helps hunters plan their gear based on weather conditions and hunting style
(e.g., stalking vs. sitting in a blind). It pulls weather data for saved locations, recommends clothing and
required equipment, and allows post-hunt reports to fine-tune comfort levels over time.

## Tech Stack (Planned)

- Java (Android) (possibly Kotlin in future)
- Retrofit or HttpClient for weather APIs
- Firebase Authentication & Cloud Firestore
- App Widgets for low-power home screen summary

_Work in progress._
